Privacy Policy

Effective Date: July 15, 2026. We respect your privacy and process zero tracking data.

1. Information We Collect

QuickShare is built with a minimal-data approach. We do not require usernames, passwords, emails, or phone numbers. The only data we actively process includes:

  • Snippet Content: The raw text or code you voluntarily copy, paste, and save on our platform.
  • Selection Choices: Selected computer language classifications and expiration limits.
  • Server logs: Standard web request headers (e.g. client IP address, date/time, requested paths, and transfer bytes) for security monitoring, spam prevention, and rate-limiting enforcement.

2. Data Storage and Retention

All shared code snippets are recorded securely inside server files (JSON format) and do not connect to permanent, unmonitored relational databases. Your snippet is retained precisely for the duration selected during creation (10 minutes, 1 hour, 1 day, 1 week) or until manual removal. Expired snippets are deleted permanently from our server systems and cannot be recovered.

3. Third-Party Services and Advertisements

We partner with Google AdSense to display clean, non-intrusive advertisements across the platform. Google may collect user browser telemetry, screen sizes, or basic geographic hints to serve tailored contextual advertisements. Google uses cookies to serve ads based on prior website visits.

You may opt-out of personalized advertising by visiting Google Ads Settings.

4. Data Security

We deploy modern industry-standard security headers, cross-site scripting (XSS) protections, cryptographic secure ID generation, and active client request rate-limiting to protect all stored files against unauthorized access, leaks, or server abuse.
